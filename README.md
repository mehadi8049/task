

## A task for execution

The application has built with laravel.

## Need for Laravel (Backend)

To begin with, You need to clone project and entry into task:

```sh
cd backend
```

```sh
composer install
```

```sh
cp .env.example .env
```
```php
php artisan key:generate
```

```sh
php artisan migrate
```

```sh
php artisan db:seed
```


```sh
php artisan serve
```
